# A'eon's Maple Mono
My customized version of [Maple Mono](https://font.subf.dev/en/).

## Use

1. Go to the Maple Mono [playground page](https://font.subf.dev/en/playground/).
2. Click the 'Generate Config' button. A popup will open.
3. Copy the contents of [`config.json`](https://github.com/aeon-tia/my-maple-mono/blob/main/config.json) into the popup's 'config.json' text field.
4. Paste the contents of [`cli-flags.txt`](https://github.com/aeon-tia/my-maple-mono/blob/main/cli-flags.txt) into the popup's 'CLI Flags' text field.
5. Close the popup.
6. Click the 'Custom Build' button to run the generator.
