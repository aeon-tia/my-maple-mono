# A'eon's Maple Mono
My customized version of [Maple Mono](https://font.subf.dev/en/).

## Use

### Playground GUI

1. Go to the Maple Mono [playground page](https://font.subf.dev/en/playground/).
2. Use the page's GUI to select the options corresponding to [`config.json`](https://github.com/aeon-tia/my-maple-mono/blob/main/config.json).
3. Click the 'Generate Config' button. A popup will open.
4. Verify that the content of the popup's 'config.json' text field matches the contents of [`config.json`](https://github.com/aeon-tia/my-maple-mono/blob/main/config.json).
5. Verify that the content of the popup's 'CLI Flags' text field matches the contents of [`cli-flags.txt`](https://github.com/aeon-tia/my-maple-mono/blob/main/cli-flags.txt).
6. Close the popup.
7. Click the 'Custom Build' button to run the generator.

### Manual builds

You can use [`config.json`](https://github.com/aeon-tia/my-maple-mono/blob/main/config.json) with Maple Mono's [GitHub Actions](https://github.com/subframe7536/maple-font?tab=readme-ov-file#2-use-github-actions), [Docker](https://github.com/subframe7536/maple-font?tab=readme-ov-file#3-use-docker), or [Local](https://github.com/subframe7536/maple-font?tab=readme-ov-file#4-local-build) build workflows directly.
